🌸 Como Instalar o App Desafio 30 Dias

📱 No Celular (Android/iOS)

Android (Chrome/Edge)

  1. Abra o site no navegador Chrome ou Edge
  2. Toque no menu (⋮) no canto superior direito
  3. Selecione "Instalar app" ou "Adicionar à tela inicial"
  4. Confirme a instalação
  5. O app aparecerá na sua tela inicial como um app normal!

iOS (Safari)

  1. Abra o site no Safari
  2. Toque no botão de compartilhar (□↑) na parte inferior
  3. Role para baixo e toque em "Adicionar à Tela de Início"
  4. Edite o nome se desejar e toque em "Adicionar"
  5. O ícone aparecerá na sua tela inicial!

💻 No Computador (Desktop)

Chrome/Edge/Brave

  1. Abra o site no navegador
  2. Clique no ícone de instalação (⊕) na barra de endereço
  3. Ou vá em Menu → "Instalar Desafio 30 Dias"
  4. Confirme a instalação
  5. O app abrirá em uma janela própria!

🚀 Como Hospedar (para funcionar online)

Opção 1: GitHub Pages (Grátis e Fácil)

  1. Crie uma conta no GitHub
  2. Crie um novo repositório público
  3. Faça upload dos arquivos: index.html, manifest.json, sw.js
  4. Vá em Settings → Pages
  5. Selecione a branch "main" e clique em Save
  6. Seu site estará em: https://seu-usuario.github.io/nome-repo

Opção 2: Netlify/Vercel (Grátis, Mais Rápido)

  1. Acesse Netlify ou Vercel
  2. Faça login com GitHub
  3. Arraste a pasta com os arquivos para fazer deploy
  4. Pronto! Você terá um link tipo https://seu-app.netlify.app

Opção 3: Testar Localmente

Para testar no seu computador antes de hospedar:

  1. Instale o Python (se não tiver)
  2. Abra o terminal na pasta do projeto
  3. Execute: python -m http.server 8000
  4. Acesse: http://localhost:8000

🎨 Criar os Ícones PNG

Você precisa converter o icon.svg para PNG nos tamanhos 192x192 e 512x512:

  • Online: Use CloudConvert ou SVG to PNG
  • Photoshop/GIMP: Abra o SVG e exporte como PNG
  • Canva: Importe o SVG e baixe como PNG

Salve como icon-192.png e icon-512.png na mesma pasta

✅ Vantagens do PWA

  • ✨ Funciona offline após primeira visita
  • 💾 Dados salvos localmente no dispositivo
  • 📱 Ícone na tela inicial como app nativo
  • ⚡ Carregamento rápido
  • 🔒 Funciona com HTTPS (necessário para PWA)
  • 📊 Progresso mantido mesmo sem internet

❓ Problemas Comuns

Botão de instalar não aparece?

  • Certifique-se que está usando HTTPS (não HTTP)
  • Verifique se o manifest.json está acessível
  • Abra o DevTools (F12) → Application → Manifest para ver erros

Service Worker não registra?

  • Verifique o console (F12) para erros
  • Certifique-se que o sw.js está na raiz do site
  • Limpe o cache e recarregue (Ctrl+Shift+R)
